# IST-Sistemas-Operativos
Projecto da cadeira Sistemas operativos na linguagem C

O objectivo deste projeto é desenvolver um sistema para o armazenamento de dados chamado  KOS  (Key value store of the Operating Systems). O KOS utiliza um modelo lógico de organização de dados chamado chave-valor (key-value), em que cada dado é associado com o par seguinte: uma "chave" que permite identificar univocamente o dado, e um "valor". Este tipo de sistemas de armazenamento de dados são usados em vários contextos, como por exemplo no registry do Windows para guardar informações de configuração e meta-dados do sistema operativo.
